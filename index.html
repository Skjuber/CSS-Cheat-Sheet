CSS

  • CSS stands for Cascading Style Sheets

  • Cascading Style Sheets is a style sheet language

  • Initial release was in 1996

Fundamentals

  • CSS describes how HTML elements are to be displayed on screen, paper, or in other media

  • CSS saves a lot of work. It can control the layout of multiple web pages all at once

  • External stylesheets are stored in CSS files

Selectors

  • CSS selectors define the pattern to select elements to which a set of CSS rules are then applied.

  • There are a wide variety of CSS selectors available, allowing for fine-grained precision when selecting elements to style.

  • There are a few different groupings of selectors, and knowing which type of selector you might need will help you to find the right tool for the job

Box Model

  • All HTML elements can be considered as boxes

  • The CSS box model is essentially a box that wraps around every HTML element. It consists of: margins, borders, padding, and the actual content.The CSS box model is essentially a box that wraps around every HTML element. It consists of: margins, borders, padding, and the actual content.

  • Key concepts to learn are: Content, Border, Padding and Margin

Flexbox

  • Flexbox is a one-dimensional layout method for arranging items in rows or columns.

  • Flexbox is a one-dimensional layout method for arranging items in rows or columns.

  • Flexbox replaced floats and positioning as superior way of arranging items on screen

  • Awesome resource to learn flexbox: https://flexboxfroggy.com/

Grid

  • Grid, is a two-dimensional grid-based layout system that completely changes the way we design user interfaces.

  • You have to define a container element as a grid with display: grid, set the column and row sizes with grid-template-columns and grid-template-rows, and then place its child elements into the grid with grid-column and grid-row.

  • Awesome resource to learn grid: https://cssgridgarden.com/
